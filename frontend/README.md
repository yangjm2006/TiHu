## ✨ 主要功能展示顺序

### 1️⃣ 以 alice 身份登录
- 查看图书列表（11 本预置图书）
- 搜索功能：输入"科幻"或按标签搜索
- 点击图书进入详情页

### 2️⃣ 在图书详情页
- 查看评分统计与分布
- 点赞/点踩评论（支持切换和取消）
- 给图书评分（1~10）
- 发表评论和回复
- 收藏图书

### 3️⃣ 社交功能
- 点击左侧"用户主页"-> 输入用户名（如 bob）-> 查看其评论和书单
- 点击"关注列表"查看已关注的用户
- 点击"粉丝列表"查看粉丝
- 点击"私信会话"与 bob 聊天（支持发起新会话）

### 4️⃣ 个人管理
- "我的收藏"：查看已收藏的图书
- "我的书单"：创建书单、添加/移除图书
- "个人信息"：修改用户名和密码

### 5️⃣ 以 admin 身份管理
- 登出后用 admin/Admin123 重新登录
- 左侧会多出一个"管理员"菜单
- 点击"图书管理"：新增图书、删除不需要的图书
- 点击"用户封禁"：封禁用户 alice（注意 alice 之后无法登录）
- 点击"评论管理"：查看全站评论，删除不合适的评论

## ⚙️ 编译与依赖

- **Java 版本**：21
- **JavaFX 版本**：21.0.6
- **构建工具**：Maven 3.8.5+
- **IDE 推荐**：IntelliJ IDEA（已配置 JavaFX 支持）

### 清理与重新编译
```bash
mvn clean compile
```

### 查看完整日志
```bash
mvn javafx:run -X
```

## 🐛 已知事项

1. **内存存储** - 重启后数据重置（这是 mock 后端的预期行为）
2. **没有网络请求** - 所有数据在本地 MockBackendService 中处理
3. **UI 未美化** - JavaFX 默认主题，后期可添加 CSS
4. **头像统一** - V1 不支持头像上传，全站使用默认头像

## 📞 问题排查

### 运行时找不到 FXML 文件
- 确保文件在 `src/main/resources/com/tihu/frontend/` 目录下
- 运行 `mvn clean compile` 后重试

### 编译错误
- 确保 Java 版本 ≥ 21
- 运行 `mvn clean compile` 清除缓存

### 登录失败
- 检查用户名和密码是否正确（默认账号见上表）
- 确认密码包含数字和英文字符
