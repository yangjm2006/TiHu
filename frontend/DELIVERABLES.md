### 认证与用户管理（4 个 Controller + 4 个 FXML）
1. **LoginController + login-view.fxml** - 登录页
   - 支持普通用户/管理员登录
   - "注册普通用户"和"注册管理员"入口
   - 测试账号提示

2. **RegisterController + register-view.fxml** - 普通用户注册
   - 用户名验证（2~10 字符，唯一）
   - 密码验证（6~12 字符，含数字和英文）

3. **AdminRegisterController + admin-register-view.fxml** - 管理员注册
   - 邀请码验证（123456）

4. **ProfileEditController + profile-edit-view.fxml** - 个人信息修改
   - 修改用户名
   - 修改密码
   - V1 头像为默认，不支持上传

### 导航与主页面（2 个 Controller + 2 个 FXML）
1. **MainController + main-view.fxml** - 主应用框架
   - BorderPane 布局（顶/左/中/底）
   - 左侧 15+ 功能菜单
   - 管理员菜单自动显示/隐藏
   - 搜索框、用户状态、登出按钮
   - 动态内容区域（StackPane）

2. **HomeController + home-view.fxml** - 首页
   - 欢迎页面，功能概述

### 图书浏览与详情（2 个 Controller + 2 个 FXML）
1. **BookListController + book-list-view.fxml** - 图书列表页
   - 支持 3 种搜索方式：
     * 书名模糊搜索
     * 标签 AND 多选搜索
     * 按评分排序（从高到低）
   - 分页（10 条/页）
   - TableView 展示（书名、作者、评分、标签摘要）
   - 双击或按钮进入详情
   - 页码信息显示

2. **BookDetailController + book-detail-view.fxml** - 图书详情与评论
   - 书名、作者、标签、简介展示
   - 评分汇总：
     * 平均分（保留 1 位小数或"暂无评分"）
     * 评分人数
     * 1~10 分布直方图
   - 我的评分（Spinner 1~10 可修改）
   - 评分提交功能
   - 一级评论与二级回复展示
   - 点赞/点踩投票（三态互斥：赞/踩/无）
   - 评论输入与提交
   - 回复一级评论功能
   - 撤回/删除评论（普通用户撤回自己，管理员删除任意）
   - 收藏/取消收藏按钮
   - 返回列表功能

### 收藏与书单（3 个 Controller + 3 个 FXML）
1. **FavoritesController + favorites-view.fxml** - 我的收藏
   - 表格展示收藏的图书
   - TableView（书名、作者、评分）
   - 查看详情按钮
   - 收藏统计

2. **BookListsController + book-lists-view.fxml** - 书单管理
   - 创建书单（标题+简介）
   - 书单列表（含书籍数量）
   - 打开书单详情
   - 删除书单

3. **BookListDetailController + book-list-detail-view.fxml** - 书单详情
   - 显示书单基本信息
   - 列表展示已加入的书籍
   - 按 ID 添加新书籍
   - 移除选中书籍
   - 返回书单列表

### 用户与社交（4 个 Controller + 4 个 FXML）
1. **UserProfileController + user-profile-view.fxml** - 用户主页
   - 支持查看任意用户（输入用户名）
   - 展示用户的评论列表（倒序）
   - 展示用户的公开书单列表
   - 显示关注数/粉丝数
   - 关注/取关按钮
   - 关系状态显示

2. **FollowingController + following-view.fxml** - 关注/粉丝列表
   - 通过参数切换：关注列表或粉丝列表
   - ListView 展示用户名列表

3. **ConversationsController + conversations-view.fxml** - 私信会话列表
   - 显示所有会话（按最后消息时间倒序）
   - 会话预览（对方用户名、最后消息内容、时间）
   - 打开选中会话
   - 发起新会话（输入用户名）

4. **ChatController + chat-view.fxml** - 聊天窗口
   - 显示完整对话历史（时间戳）
   - 消息列表（从/到/内容）
   - 纯文本消息输入
   - 发送消息
   - 返回会话列表

### 管理员后台（3 个 Controller + 3 个 FXML）
1. **AdminBooksController + admin-books-view.fxml** - 图书管理
   - 新增图书（书名必填且唯一）
   - 编辑字段（作者、简介、标签）
   - 逻辑删除图书
   - TableView 展示所有图书（书名、作者、评分）

2. **AdminUsersController + admin-users-view.fxml** - 用户管理
   - 封禁用户（指定小时数）
   - 解封用户
   - 显示被封禁用户列表（按到期时间排序）
   - 自动检查过期

3. **AdminCommentsController + admin-comments-view.fxml** - 评论管理
   - 显示全站所有评论（倒序）
   - 逻辑删除评论
   - 前端自动不显示已删除评论

### 业务逻辑（2 个核心类）
1. **MockBackendService.java** - 完整 mock 后端
   - 用户认证与权限管理
     * 登录、用户注册、管理员注册（邀请码）
     * 用户名/密码规则验证
     * 用户头像系统（默认）
   - 图书管理
     * 图书增删改查
     * 书名唯一性检查
     * 标签解析与存储
   - 评分系统
     * 每用户每书评分一次（可修改）
     * 平均分计算
     * 分布统计
   - 评论与回复
     * 两级结构（一级+二级回复）
     * 最多 200 字
     * 时间戳自动生成
     * 用户撤回/管理员删除
   - 投票系统
     * 赞/踩/无三态互斥
     * 支持取消和切换
   - 收藏系统
     * 收藏/取消收藏
     * 按用户查询收藏列表
   - 书单管理
     * 创建/编辑/删除书单
     * 加书/移除书
     * 同一书单中同一书只能一次
     * 全部公开
   - 关注系统
     * 单向关注（我关注你）
     * 关注数/粉丝数统计
     * 关注/取关功能
   - 私信系统
     * 一对一聊天
     * 无需关注
     * 消息持久化（在内存中）
     * 会话自动排序
   - 用户管理
     * 用户封禁（指定到期时间）
     * 登录时检查封禁
   - 默认测试数据
     * 11 本预置图书（带标签和简介）
     * 3 个测试用户（admin/alice/bob）
     * 评分数据、评论数据、关注关系、私信等

2. **ApiClient.java** - HTTP 客户端（预留）

## 🔑 关键特性

### 会话管理
- AppContext 单例存储登录用户
- 页面导航时自动传递 MainController 引用
- 所有操作都经过权限检查

### 数据流向
```
UI 操作 → Controller → Service → AppContext/MockBackendService → 数据更新 → UI 刷新
```

### 安全特性
- 用户名唯一性检查（大小写敏感）
- 密码规则验证（数字+字母）
- 用户权限隔离
- 管理员邀请码验证
- 用户封禁时登录阻止

### 逻辑删除
- 图书删除后关联数据不可见
- 评论删除后前端过滤不显示
- 数据保留用于审计/恢复
